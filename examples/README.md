# go sdk examples

这些 examples 旨在帮助您快速了解使用又拍云 go-sdk。

这些 examples 都可以直接运行，但是在运行之前，需要根据您的需求填写相应的参数。

### 参数：

#### 用户需先配置个人信息环境变量

`UPYUN_BUCKET` 您的空间名

`UPYUN_USERNAME` 您的操作员名

`UPYUN_PASSWORD` 您的用户密码

#### 使用和测试实例时

用户个人参数见 `user_profile.go` 和又拍云文档说明

举例说明：

`NOTIFY_URL` 您的通知URL

`LOCAL_IMAGE` 您的图片文件本地存放路径
